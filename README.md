# Calorie-Calculator
MERN full stack app that allow you to track total calories you have today

```sh
cd ./frontend
npm install
npm start
cd ./backend
npm install
npm run dev
```
*Creat a .env file and paste your MongoDB URI
```sh
mongoURI=
```
